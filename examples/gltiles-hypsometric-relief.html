--- layout: example.html title: WebGL hypsometric tint shortdesc: WebGL shader for Terrain-RGB hypsometric tint and shaded relief docs: >

Similar to the gltiles-shaded-relief example, uses an eleation model to calculate both a hypsometric tint and a shaded relief.

This loads elevation data from MapTiler's Terrain-RGB tiles, then calculates both the colour for the hypsometric tint and the shaded relief parameters. All the heavy lifting of pixel algebra is done in the GPU via WebGL.
